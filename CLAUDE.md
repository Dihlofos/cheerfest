## Команды проекта
- run: `npm run start`

## Договорённости
- не трогай папку build, все изменения в папке source
- Вся разметка лежит в файле index.html, когда я проше что - то поменять в разметке - надо смотреть туда
- если я прошу тебя "сбрось кеш" - это значит, что нужно пойти в файл gulpfile.js, и итерировать название файла main.js на 97й строке (если был main.js, сделать main1.js), и тоже самое с названием файла style.min.css на строке 49. Так же итерировать это название во всех html страничках проекта
- При написании нового кода в html нужно придерживаться принципов БЭМ - блок элемент модификатор. Для каждого блока следует создавать отдельный файл .scss
- В файле .scss где описаны стили блока не должно быть стилй другого блока. Один файл - один блок.
- дизайн сайта лежит в файле design.jpg
- Все размеры - отступы, типографика и т.д. нужно писать в единицах rem
- Все цвета нужно брать из переменных, файл colors.scss
- Все шрифты подключаем через переменныей, файл vars.scss
